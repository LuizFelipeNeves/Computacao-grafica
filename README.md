# Django Simple Detect Number Plate

## Running Locally

```bash
git clone https://github.com/LuizFelipeNeves/Simple-Number-Plate-Detect.git
```

```bash
pip install -r requirements.txt
```

```bash
python manage.py migrate
```

```bash
python manage.py runserver
```
